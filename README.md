Project Pohon Keluarga merupakan sebuah aplikasi berbasis web yang dikembangkan menggunakan framework Laravel dengan tujuan untuk mengelola dan menampilkan data silsilah keluarga secara terstruktur dan sistematis. Aplikasi ini memungkinkan pengguna untuk menyimpan informasi anggota keluarga, seperti nama, jenis kelamin, dan tanggal lahir, serta menghubungkan setiap anggota dengan orang tua mereka untuk membentuk hubungan antar generasi.

Dalam project ini, konsep relasi data parent–child diterapkan pada database, sehingga struktur keluarga dapat ditampilkan dalam bentuk pohon keluarga. Setiap anggota keluarga dapat memiliki satu orang tua dan beberapa anak, yang mencerminkan hubungan keluarga secara hierarkis. Pengelolaan data dilakukan melalui fitur CRUD (Create, Read, Update, Delete) yang memungkinkan pengguna untuk menambah, melihat, mengubah, dan menghapus data anggota keluarga dengan mudah.

Aplikasi Pohon Keluarga dibangun dengan arsitektur MVC (Model–View–Controller) yang menjadi standar pada Laravel. Model digunakan untuk mengelola data dan relasi database, View untuk menampilkan antarmuka pengguna, dan Controller sebagai penghubung antara logika aplikasi dan tampilan. Database yang digunakan dapat berupa MySQL maupun SQLite, tergantung pada kebutuhan pengembangan.

Project ini bertujuan untuk memberikan pemahaman tentang penerapan relasi database, penggunaan migration Laravel, serta implementasi sistem manajemen data berbasis web. Selain itu, project Pohon Keluarga dapat dikembangkan lebih lanjut dengan penambahan fitur seperti autentikasi pengguna, visualisasi pohon keluarga secara grafis, dan ekspor data ke dalam format laporan.
